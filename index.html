How it works

Simple workflow for chemical upload preparation

01

Prepare Data Entry

Add CAS numbers and any manual fields from Quartzy, vendor records, or lab notes.

02

Fetch PubChem data

The app pulls available fields such as SMILES, formula, molecular weight, IUPAC name, and InChIKey.

03

Download output

You receive a clean Excel file with Data Entry and CDD Upload sheets ready for review.

Input requirements

  • File type must be .xlsx
  • Sheet name must be Data Entry
  • Column CAS Number is required

Important check

PubChem lookup can return a different salt form, parent compound, or synonym match. Always review SMILES, salt, and compound identity before final CDD upload.
